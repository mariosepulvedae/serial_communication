### Serial communication with java

To run this project you just need to run this command
```
javac javac -cp ".:./jSerialComm-X.X.X.jar" Serial.java
```
Then, you can run the whole application with the next command:
```
java -cp ".:./jSerialComm-X.X.X.jar" Serial
```
where ```X.X.X``` refers to the current version of the jSerialComm library.

You can check the jSerialComm documentation [here](https://fazecast.github.io/jSerialComm/)
